# lspee

This is a minimal reservation crate used to preserve the `lspee` package name.

For production usage, install and run the CLI package:

```bash
cargo install --path ../lspee_cli
```

Or use the implementation crates directly:

- `lspee_cli`
- `lspee_daemon`
- `lspee_lsp`
- `lspee_config`
- `lspee_protocol`
